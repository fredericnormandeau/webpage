# Frederic Normandeau - Personal Website

Classic black-and-white one-page personal website.

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Notes

- The layout is intentionally classic and minimal (black and white).
- The header includes personal profile links:
  - Website (current page)
  - LinkedIn: `https://www.linkedin.com/in/fredericnormandeau/`
  - GitHub: `https://github.com/fredericnormandeau`
- Update section content directly in `index.html`.
